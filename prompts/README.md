# Prompts

AI 도구에 입력한 프롬프트를 기록하여 재현 가능성을 보장합니다.

## 목적

- **재현성**: 동일한 프롬프트로 시간대별 AI 성능 비교
- **공정성**: 모든 AI 도구에 동일한 정보와 지시 제공
- **학습**: 효과적인 프롬프트 패턴 발견 및 개선

## 파일 형식

각 구현마다 하나의 마크다운 파일:

```
{ai-tool}-{model}-{YYYYMMDD}.md
```

### 예시
- `claude-sonnet-4.5-20260214.md`
- `gpt-4-20260215.md`
- `cursor-20260220.md`

## 프롬프트 기록 템플릿

```markdown
# {AI Tool} {Model} - {Date}

## 초기 프롬프트
[프로젝트 설명 및 첫 번째 요청]

## 대화 기록

### Turn 1
**User:**
[사용자 입력]

**AI:**
[AI 응답 요약 - 생성된 파일 목록]

### Turn 2
...

## 총평
- 총 대화 턴 수: X
- 소요 시간: Y시간
- 주요 이슈: [발생한 문제 및 해결 과정]
- AI 제안사항: [AI가 제안한 개선점]
```

## 프롬프트 템플릿 (재사용용)

`prompt-templates.md`에 효과적인 프롬프트 패턴 저장:
- 초기 프로젝트 설정 프롬프트
- 스펙 전달 프롬프트
- 디버깅 요청 프롬프트
- 코드 리뷰 요청 프롬프트
